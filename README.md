## 编译方法

运行命令下载依赖，自行添加镜像源要不然会很慢

```
pip install -r requirements.txt
```

如果pip不能下载torch，建议使用anaconda下载

```
conda install pytorch torchvision torchaudio cpuonly -c pytorch
```

## 使用方法

进入031902332文件夹下，命令行运行下面命令自动执行app.py文件

```
flask run
```
